# 동키 (동네 위키)

---

## Introduce

사용자가 직접 동네 가게(Ex. 음식점)를 등록하고, 판매 상품, 가격 등의 정보를 편집 할 수 있는 플랫폼입니다.

사용자의 권한에 따라 편집에 제한이 있을 수 있으며, 부적절한 편집에 대해서는 강력한 제제를 통해 위키 생태계를 유지하고자 합니다.
